# Sistema de Monitoramento de Encostas com IoT

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de monitorar áreas de risco de deslizamentos utilizando Internet das Coisas (IoT). O sistema realiza o monitoramento da intensidade da chuva e da distância do solo, permitindo identificar situações de atenção e perigo.

Quando um risco é detectado, o sistema aciona alertas visuais e sonoros, envia notificações automaticamente pelo Telegram e disponibiliza um painel web para acompanhamento em tempo real.

## ⚙️ Funcionalidades

* Monitoramento da intensidade da chuva;
* Monitoramento da distância utilizando sensor ultrassônico;
* Classificação dos níveis de risco (Seguro, Atenção e Perigo);
* Acionamento de LEDs indicadores e buzzer;
* Envio de alertas automáticos pelo Telegram;
* Painel web com atualização em tempo real;
* Comunicação entre Arduino e ESP8266.

## 🛠️ Tecnologias utilizadas

* Arduino
* ESP8266 (NodeMCU)
* C++
* HTML
* Google Charts
* Telegram Bot API
* Wi-Fi

## 🎯 Objetivo

Desenvolver uma solução de baixo custo para auxiliar no monitoramento de áreas suscetíveis a deslizamentos, permitindo que alertas sejam enviados rapidamente aos responsáveis e contribuindo para a prevenção de acidentes.
